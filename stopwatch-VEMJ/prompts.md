Genera el código para un StopWatch en JavaScript. Los requerimientos son estos:

- Botón Start
Cuando se haga clic en el, correrá el tiempo y se cambiará el texto del botón por "Pause".
Cuando se haga clic de nuevo en el botón, el texto de "Pause" cambiará a "Continue" y el StopWatch se detendrá.
Cuando se haga clic de nuevo en el botón, el texto de "Continue" cambiará a "Pause" y el StopWatch continuará. 


- Botón Clear
Pone el StopWatch a cero, es un reinicio de tiempo.

- Etiqueta
Una etiqueta o label HTML que marque el tiempo del StopWatch en horas, minutos, segundos y milisegundos.

Genera tres archivos uno que sea el HTML (index.html), uno de JavaScript (scripts.js) y otro que tenga los estilos CSS (style.css). Utiliza algún framework como Bootstrap para darle una apariencia amigable. Haz que los controles HTML sean más grandes de lo normal para interactuar fácilmente en la pantalla con ellos.
